dogshaming
==========

hubot script that pulls a random dog from dogshaming.com

Description:
  Hubot delivers a freshly shamed dog.

Dependencies:
  - nodepie: 0.5.x
  - node-expat: 2.0.x

Configuration:
  None

Command(s):
  shame a dog - Gets random pic from dogshaming.com
